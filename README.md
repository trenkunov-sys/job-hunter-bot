# 🤖 Job Hunter Bot — Production Edition

Telegram-бот для поиска работы за границей с AI-анализом резюме и мульти-источниками вакансий.

## 🚀 Возможности

### 📄 Работа с резюме
- Приём PDF, DOCX, TXT
- Проверка файлов (магические байты, лимит 5 МБ)
- AI-определение профессии (25+ специальностей)

### 🔍 Источники вакансий
| Источник | Статус | Ключ | Описание |
|----------|--------|------|----------|
| **Himalayas** | ✅ Работает | Не нужен | Удалённые вакансии, стартапы |
| **Adzuna** | 🔑 Опционально | Нужен | Европа, США, Канада, Австралия |
| **SerpAPI** | 🔑 Опционально | Нужен | Google Jobs (LinkedIn, Indeed, Glassdoor) |
| **Mock** | ✅ Всегда | Не нужен | Демо-вакансии для теста |

### 🧠 AI-функции (DeepSeek)
- Генерация сопроводительных писем
- A-F Scoring — анализ соответствия вакансии
- ATS-оптимизация резюме
- Rate Limiting: 5 запросов/мин

### 🔔 Дополнительно
- Подписка на рассылку вакансий
- Сохранение избранного в SQLite
- Статистика использования
- Обратная связь разработчику (@Trene4ca)
- Команда `/forget_me` — удаление данных (GDPR)
